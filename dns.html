DNS Change



On June 1, 2023, RiiConnect24 will be deprecating the old server used to provide DNS services (164.132.44.106) and you can no longer connect to RiiConnect24 using that DNS. Additionally, that DNS is used by some people to enjoy games online using Wiimmfi using Wii or DS. You will have to change the DNS on that as well. In order to continue using RiiConnect24, you must change your DNS to the following on your Wii and/or DS...


Primary DNS: 167.86.108.126
Secondary DNS: 1.1.1.1

Thank you for the continued support of RiiConnect24.
