RiiConnect24 Bot

Look up error codes and exchange friend codes with other users.
Invite to Server


RiiConnect24 Bot is a Discord bot that's written in Java that allows you to look up error codes for Wiimmfi or RiiConnect24. It's powered by the error API from Wiimmfi. You can also exchange codes with other users for the Wii (U), Switch, 3DS, and game-specific codes.
The source code for RiiConnect24 Bot is on GitHub. After you invite the bot to your server, you can find all the commands you can use by typing in "!help" in the chat. You can always invite the bot to your server by going to bot.rc24.xyz.
